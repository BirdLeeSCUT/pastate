## 贡献指引
1. fork pastate 的 github 仓库： https://github.com/BirdLeeSCUT/pastate
2. clone 并安装 npm 包
```bash
$ git clone <your repo>
$ cd pastate
$ yarn install
```
3. 运行 pastate 开发包
```bash
$ yarn start
```
4. 实时测试 pastate 开发包
```bash
$ yarn test
```
5. 编译 pastate 开发包
```bash
$ yarn dist
```
目标代码会编译到 /dist 目录下
6. 提交代码，发起 pull request。
